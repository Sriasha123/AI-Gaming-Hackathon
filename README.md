# AI Gaming Hackathon Platform

A full-stack web application for managing and hosting AI Gaming Hackathons. The platform enables organizers to handle registrations, payments, team management, project submissions, announcements, analytics, and participant engagement through dedicated dashboards.

## Key Features

* Interactive landing page with countdown timer
* Team registration and participant management
* Payment workflow with receipt upload support
* Automated team ID generation and email notifications
* Participant dashboard for project submissions and updates
* Admin dashboard for registration monitoring and management
* Announcement and communication system
* Registration analytics and reporting
* CSV export functionality
* Responsive and modern user interface

## Tech Stack

### Backend

## Objective

To provide a centralized platform that simplifies hackathon management by automating registrations, communications, project submissions, analytics, and administrative workflows.
